The site is organized as follows:


- [Overview](../overview/index.md): This page describes what *ODM Platform* is about, its goals and its scope. Start from here before deep diving into other parts of the documentation.
- [Concepts](../concepts/index.md): This page describes the key concepts that inspired the design of the *ODM Platform*.
- [Architecture](../architecture/index.md): This page provides a deep dive into the architectural components of the *ODM Platform*.
- [Integration Hub](../integration-hub/index.md): This page provides a list of adapter services available to integrate the *ODM Platform* with data infrastructure tools or to extend it by developing new integrations.
- [API Doc](../api-doc/index.md): This page provides links to explore the REST APIs exposed by the *ODM Platform* services.
- [Community](../community/index.md): This page explains the possible ways to contribute to the evolution of the *ODM Platform*.
- [Resources](../resources/index.md): These pages provide useful resources to explore the role of the *ODM Platform* as an XOps platform.
