Data Mesh Resources⚓︎

Zhamak's Corner⚓︎

Zhamak Dehghani :fontawesome-brands-linkedin: formalized the data mesh approach to data management in 2019 while working at Thoughtworks as the director of emerging technologies. Since then Zhamak has continued to work to disseminate and evolve the data mesh approach within the data and business community. Today she is the CEO and founder of a stealth startup and undoubtedly one of the most important references in the data mesh world.

Resource Date Description
How to Move Beyond a Monolithic Data Lake to a Distributed Data Mesh :octicons-link-external-24: May 2019 This is the first Zhamak’s data mesh article posted on Martin Fowler’s blog. She focuses here on recurrent problems in data management and propose data mesh as a new apprach to solve them.
Data Mesh Principles and Logical Architecture :octicons-link-external-24: December 2020 This is the second Zhamak’s article on Martin Fowler’s blog where she delves deeper into data mesh approach and its key priciples.
Introduction to Data Mesh :octicons-link-external-24: June 2022 This is a more recent presentation made by Zhamak at the Stanford Deep Data Research Computing Center. The concepts are the same of the previous two articles but they are refined and extended.
The four principles of Data Mesh :octicons-link-external-24: NA This is a series of webinars, each one focusing on one of the data forur data mesh key principles.
Data Mesh :octicons-link-external-24: March 2022 This book it is as for now the ultimate reference for the data mesh apprach to data management.
Software Architecture: The Hard Parts :octicons-link-external-24: October 2021 In this book Zhamak curates the parts related to data architectures.

Communities⚓︎

Resource Description
Data Mesh Learning Community :octicons-link-external-24: This is a vendor-independent learning resource created to teach people about data mesh. This website centralize data mesh content and address common data mesh questions and misconceptions. There is also a Slack channel :octicons-link-external-24: associated with the site to discuss about data mesh topics with other community members.

Podcasts⚓︎

Resource Description
Data Mesh Radio Podcast :octicons-link-external-24: This podcast hosted by Scott Hirleman :fontawesome-brands-linkedin:, the co-founder of the Data Mesh Learning Community :octicons-link-external-24:, focuses on topics that are near and dear to those implementing – or considering implementing – data mesh.
Data Engineering Podcast :octicons-link-external-24: This podcast hosted by Tobias Macey :fontawesome-brands-linkedin: tackles a new approach to data management every week. Each new episode provides useful and informative insights into the projects, platforms, and practices that data engineers, team leaders, and data scientists need to know about to learn and grow in their career.
Catalogue & Cocktails Podcast :octicons-link-external-24: This podcast hosted by Juan Sequeda :fontawesome-brands-linkedin: and Tim Gasper :fontawesome-brands-linkedin: is an honest, no-bs, non-salesy conversation about enterprise data management and analytics.

Other data mesh resources⚓︎

Resource Description
Data Mesh Architecture :octicons-link-external-24: This website is a nice 360 degree intro to data mesh from an engineering prespective.
State of Data Mesh 2022 :octicons-link-external-24: This website contains the video recording of the sections of last State Data Mesh conference (2022).
